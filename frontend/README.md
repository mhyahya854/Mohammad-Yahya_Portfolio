# Mohammad Yahya Portfolio

A modern, responsive portfolio website showcasing projects, skills, education, and experience built with React, TypeScript, and Tailwind CSS.

## Overview

This is the frontend application for Mohammad Yahya's professional portfolio. It features a beautiful space-themed design with smooth animations and interactive elements.

## Repository

- **GitHub**: [https://github.com/mhyahya854/Mohammad-Yahya_Portfolio](https://github.com/mhyahya854/Mohammad-Yahya_Portfolio)

## Features

- Responsive design for desktop and mobile
- Hero section with space background
- About, Skills, Projects, Education, and Contact sections
- Dark theme with celestial aesthetic
- Smooth scrolling and animations
- Resume integration

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Shadcn UI Components
- Vitest for testing

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun package manager

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will start at `http://localhost:3000`

### Build

```bash
npm run build
```

### Testing

```bash
npm run test
npm run test:watch
```

## Deployment

This project is deployed on Vercel. See the [vercel.json](../vercel.json) configuration file for deployment settings.
